You can view a project's source code in your prefered internet browser.


            For Microsoft Edge users:

                1. Navigate to the "Settings and More" menu."

                2. Click on the "F12 Developer Tools" option.

            For Internet Explorer users:

                1. Press the "alt" key to make the file menu visible if it isn't already.

                2. Navigate to the "View" menu, and click on "Source."

                or:

                1. Right-click on the website page.

                2. Click on "View Source."

            For Mozilla Firefox users:

                1. Press the Alt key to bring up the browser's menu bar if it isn't already visible.

                2. Select Tools, Web Developer, and then Page Source.

            For Google Chrome users:

                1. Click the menu icon "Menu" on the browser toolbar.

                2. Select More tools, then click on "View."

            For Opera users:

                1. From the menu, go to "View."

                2. Click on "Developer Tools" then select "Source."
